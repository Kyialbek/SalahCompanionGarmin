# Installing Salah Companion on Garmin

## File to share

Send this final signed Connect IQ package to friends:

```text
bin/SalahCompanion.iq
```

## Private sharing options

You can send the `.iq` file privately using:

- WhatsApp
- Telegram
- Email
- Google Drive
- AirDrop

## Installation note

Friends can receive the `.iq` file, but installing it on a real Garmin watch may require Garmin Express on Mac or Windows, or another Garmin-supported sideload method for their specific watch model.

Typical manual testing flow:

1. Receive `SalahCompanion.iq`.
2. Connect the Garmin watch to a Mac or Windows computer.
3. Open Garmin Express or use the supported Garmin sideload workflow for that watch.
4. Install or copy the app package using Garmin's supported process.
5. Disconnect the watch and open Salah Companion from the watch app list.

## iPhone note

For iPhone users, WhatsApp, Telegram, AirDrop, Email, or Google Drive are fine for sending the file. Installation on the actual Garmin watch may still require Garmin Express on a computer because iOS does not generally provide direct Connect IQ sideload installation from a chat attachment.

## Recommendation

For easier sharing with many users, prepare the app for a Garmin Connect IQ private beta later. A private beta or store listing is the cleanest path for non-technical testers because Garmin handles device compatibility and installation through the official Connect IQ flow.
